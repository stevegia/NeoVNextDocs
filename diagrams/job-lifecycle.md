## 2. Job State Machine

```mermaid
stateDiagram-v2
    [*] --> queued : POST /api/jobs → INSERT job_queue
    queued --> assigned : Go Supervisor claim_next_job()
    assigned --> running : Supervisor set_job_running()
    running --> completed : Supervisor complete_job()<br/>outputs uploaded + registered
    running --> failed : Handler error / timeout
    assigned --> queued : Lease expired (10 min)

    completed --> synced : ArtifactSyncService<br/>File.Copy + INSERT job_artifacts
    failed --> [*]
    synced --> [*]

    note right of assigned
        Lease: 10 minutes (LEASE_DURATION_MINUTES).
        Lease renewer goroutine extends during processing.
        Expired leases reclaimed on next poll.
    end note

    note right of completed
        ArtifactSyncService polls every
        ClusterPollingIntervalSeconds (10s).
        Copies outputs from io-cache to artifact-data.
    end note
```

## 4. Key Configuration

| Config Key | Service | Default | Purpose |
|---|---|---|---|
| `Gpu:Enabled` | Backend | `true` | Enable GPU job dispatch |
| `Gpu:Platform` | Backend | `RunPod` | Platform adapter: `RunPod` or `GoogleCloud` |
| `Gpu:RunPod:PodId` | Backend | — | RunPod pod ID (direct) |
| `Gpu:RunPod:PodName` | Backend | — | RunPod pod name (resolved via API) |
| `Gpu:RunPod:ApiKey` | Backend | — | RunPod API key for pod management |
| `Gpu:RunPod:WorkerToken` | Backend | — | Auth token for CacheTransferController |
| `Gpu:BudgetCents` | Backend | `500` | Monthly GPU budget cap |
| `POLL_INTERVAL_SECONDS` | Go Supervisor | `3` | Job polling frequency |
| `LEASE_DURATION_MINUTES` | Go Supervisor | `10` | Job lease timeout |
